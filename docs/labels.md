# Labels

Recommended GitHub labels for this repository:

- `bug` - incorrect behavior or regressions
- `enhancement` - user-visible improvements
- `docs` - documentation-only changes
- `good first issue` - small, well-scoped work
- `help wanted` - items suitable for contributors
- `ui` - layout, styling, or presentation changes
- `gameplay` - rule or progression changes
- `tests` - test coverage or validation updates
